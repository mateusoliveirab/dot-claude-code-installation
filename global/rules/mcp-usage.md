# MCP Usage

The following MCP servers are available and should be used when appropriate:

## Chrome DevTools
- **When to use:** To validate UI changes, inspect elements, verify layout and visual behavior.
- **Examples:** Test if a component rendered correctly, verify responsiveness, inspect applied styles.
- **Shortcuts:** When user says "check UI", "test interface", "validate layout"

## Shadcn UI
- **When to use:** To add new UI components or look up references for existing shadcn/ui components.
- **Examples:** Add a Button, Dialog, Card or any shadcn component to the project.
- **Shortcuts:** When user says "add component", "use shadcn", "UI component"

## Supabase
- **When to use:** For database operations, authentication and storage via Supabase.
- **Examples:** Query tables, create migrations, manage schemas.
- **Shortcuts:** When user says "query DB", "database", "auth", "storage"
